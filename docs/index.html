核心功能

🎨

贴纸互动

在任意网页上拖拽贴纸（🍅🐶🤡👍❤️🔥），表达你的态度和情绪。

💬

魂系建言

组合文字评论（"前方 + 有干货"），留下你的建言和提示。

🌐

实时多人

WebSocket 实时同步，看到其他用户在同一页面上的互动。

🎵

音效反馈

每个贴纸都有独特的合成音效，沉浸式体验。

📍

智能定位

基于 DOM 元素定位，滚动时贴纸自动跟随。

⚡

极致性能

纯文本传输，单条消息 < 300 字节，2核2G即可运行。

安装教程

1

下载扩展

点击上方按钮下载 webparty_extension_v1.0.zip 并解压到本地目录。

[GIF 演示：下载并解压]
2

打开扩展页面

在 Chrome 浏览器访问 chrome://extensions/，开启右上角的"开发者模式"。

[GIF 演示：打开扩展页面并开启开发者模式]
3

加载扩展

点击"加载已解压的扩展程序"，选择解压后的文件夹。

[GIF 演示：加载扩展程序]
4

开始使用

打开任意网页，右下角会出现 WebParty 面板，开始贴贴纸吧！

[GIF 演示：使用插件]
